Github 会更新全球主要股指的情况，也会做出一些预测

基本思想是，中美是世界上先进生产力的代表，买入并长期持有是非常赚的

具体操作上，根据 RSI 周期，在绿色期买入，在红色期卖出

本项目用来快速分析股票数据，尤其是国内股票平台不常用的 RSI 数据（相当于导数）

我会每次看的时候都顺手更新一下，不过这样数据就没法实时更新

后续可能会做个网页版，用来自动同步数据，每天自动更新
